# Opportunity: Streaks

---

## What we're building

We're going to add a streak counter to the home screen. It'll show how many days in a row the user has completed a session, with a flame icon and a number. Tapping it shows a calendar of completed days. We'll send a push notification at 8pm if the user hasn't done a session yet to protect their streak.
